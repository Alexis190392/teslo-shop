# TESLO API

1. Clonar el proyecto.
2. Instalar dependencias.
```
yarn install
```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```.
4. Cambiar las variables de entorno a medida.
5. Levantar la base de datos
```
docker-compose up -d
```
